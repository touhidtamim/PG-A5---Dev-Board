Question-1: What are the different ways to select an element in the DOM?

Answer: Webpage-er kono ekta jinis ke JavaScript diye dhorte (select korte) onek gula upay ache. Nicher gulo sob theke beshi use hoy—

  1. ID use kore: Kono ekta element er unique ID thakle seta diye easily select kora jay.
  2. Class use kore: Jodi onek gulo element er same class thake, tahole shobgulo ke eksathe select kora jay.
  3. Tag name use kore: Ekoi dhoroner tag (h1, p, div etc.) sob ekbare select kora jay.
  4. Query Selector use kore: CSS er moto likhle ID, class, tag sob kichu ekta method diye select kora jay.
  5. Name use kore: Mainly form-er element gula select korar jonno use hoy.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:The difference between innerHTML, innerText, and textContent are—

Feature innerHTML innerText textContent
HTML Tag Yes, includes HTML tags No, just visible text No, just text
CSS Influence No, unaffected by CSS Yes, affected by CSS No, unaffected by CSS
Performance Slower (parses HTML) Faster than "innerHTML" Fastest (works with raw text)

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling holo ekta process jekhane, kono event (like click) apply kora hole, tokhon sei event ta parent elements gulor dike uthte thake. And shesh porjonto root element e giye theme jay. Tobe amra caile ageo thamate pari "event.stopPropagation()" method use kore.


Kibhabe kaj kore?

  • Jokhon ekta child element e event trigger hoy, seta prothome child element ei handle hoy.
  • Tarpor event ta bubble hoye tar parent element e chole jay, abar tar parent e, eirokom kore root element porjonto uthe.
  • Jodi parent elements gulo event listener set kore rakhe, tahole tarao sei event receive korte pare.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: JavaScript use kore amra dynamically elements create, add and remove korte pari.


Kibhabe kaj kore?

  1. Element Create: "document.createElement()" use kore notun ekta element create kora jay.
  2. Element Add: "appendChild()", "append()", "prepend()", "insertBefore()" use kore element ke kono parent er moddhe add kora jay.
  3. Element Remove: "removeChild()" or "remove()" use kore kono element ke DOM theke remove kora jay.
